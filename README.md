<a name="Run"></a>
## Run

1. Clone the repository:

  ```bash
  $ git clone git@github.com:iShuga139/heroes-java-spring-web-app.git
  $ cd heroes-java-spring-web-app
  ```

2. Execute the application:

  ```bash
  $ ./mvnw spring-boot:run
  ```
